# tarasister GA4 Analytics Dashboard

TARA SISTER (tarasister.com / BASE) のGA4アクセス解析ダッシュボード。

## 構成
- **Google Apps Script**: GA4 Data APIからデータ取得 → スプレッドシート書き出し + Web API (doGet)
- **GitHub Pages** (docs/index.html): ダッシュボードUI

## ダッシュボード
https://ginzasugiden.github.io/taraga4/

## レポート内容
- 日別サマリ (ユーザー数・セッション・PV・CV)
- ページ別 (上位100件)
- 集客チャネル別
- コンバージョン別 (purchase)
